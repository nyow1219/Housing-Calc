# Home Affordability Analyzer

## Overview

The Home Affordability Analyzer is an interactive web application that helps users estimate whether they can realistically afford to purchase a home based on their financial situation. By combining mortgage calculations with common lending guidelines, the calculator provides a breakdown of monthly housing costs and shows whether a home fits within a user's budget.

I built this project because housing affordability is an issue that affects many people, especially first-time homebuyers. I wanted to create a tool that makes financial information easier to understand while applying concepts from economics and software development.

---

## Features

- Calculate the maximum affordable home price
- Estimate monthly mortgage payments
- Include property taxes, homeowners insurance, PMI, and HOA fees
- Calculate debt-to-income (DTI) ratio
- Combine incomes for partners or roommates
- Adjust affordability based on credit score
- Support down payments entered as either a dollar amount or a percentage
- Display affordability using color-coded indicators
- Visualize mortgage rates and U.S. median home prices with interactive charts

---

## Why I Built It

I wanted to create a project that solves a real-world problem instead of just another basic calculator. Buying a home is one of the biggest financial decisions people make, and I wanted to build a tool that helps users better understand how income, debt, down payment, and interest rates all affect affordability.

This project also gave me experience designing an interactive application, organizing financial calculations, and creating a user-friendly interface.
